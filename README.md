# Volume-control-with-face-detection
This is an opencv programme and it is made to use with  webcam of PC
This repository consists of two file. First File is of python code . Second File is mp3 song.
For this program to work both the files should be in same directory.

Working
The program is made to work with webcam and range isn't large. Range is almost .5 to 4 feet.
Program recognises human face and sets the volume of the music in accordance to the distance between user and the webcam.
